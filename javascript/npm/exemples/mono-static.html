Faust Web Assembly Library

Mono Audio node example

Load and run the following precompiled dsp code, using: faust -lang wasm djembe.dsp -o djembe.wasm

import("stdfaust.lib");
process = ba.pulsen(1, 10000) : pm.djembe(60, 0.3, 0.4, 1) <: dm.freeverb_demo;

Javascript code

audioCtx = new (window.AudioContext || window.webkitAudioContext)();
createMonoAudioNode(audioCtx, "djembe.wasm", "djembe.json").then(node => {
    node.connect(audioCtx.destination);
});
